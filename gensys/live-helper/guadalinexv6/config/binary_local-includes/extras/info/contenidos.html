Guadalinex v6 - Software Libre

CONTENIDO DEL DISCO:

Guadalinex V6

Guadalinex es un sistema operativo y un conjunto de programas que hacen tu ordenador más util, seguro y potente. Desarrollado por la Junta de Andalucía para facilitar el acceso a todos a la Sociedad de la Información, Guadalinex es un producto libre y gratuito.

Este disco incluye el sistema completo sin limitaciones.

Suplementos para Guadalinex

Además del sistema Guadalinex básico, en este disco encontrarás una selección de aplicaciones opcionales:

  • Suplemento de Juegos: estrategia, ajedrez, simuladores... diversión para toda la familia.
  • Suplemento Educativo: programas didáctivos para niños y no tan niños.
  • Suplemento de Desarrollo: para los que sepan programar, administrar servidores o hacer páginas web (o quieran aprender).
  • Suplemento Variado: un surtido de utilidades y aplicaciones de diversos sectores.

En cualquier momento, después de instalar Guadalinex, puedes insertar de nuevo este disco y agregar nuevas aplicaciones a tu ordenador.

Software Libre para MS-Windows

Puedes usar software libre sin instalar Guadalinex. Este disco incluye una recopilación de programas para MS-Windows clasificados por temas.

Todos tienen licencias libres por lo que no tienes que pagar por usarlos y puedes compartirlos y copiarlos sin ningún impedimento.
