Estos paquetes de software no se han incluido en el disco por diversos motivos. Por favor, lea la explicación que acompaña a cada uno y decida Vd. mismo si los desea instalar.

Acrobat Reader:

Lector de archivos PDF. En el CD de Guadalinex se incluye el lector Evince, que permite ver la mayoría de ficheros PDF. Sin embargo, algunos archivos de este tipo incluyen formularios rellenables y otras características especiales que no se ven correctamente si no es con Acrobat Reader .Es un producto gratuito de Adobe pero no es software libre.

  • Instalar lector
  • Instalar plugins

Soporte para ficheros RAR:

El formato rar es un formato de compresión muy popular en Internet. Para que se puedan visualizar y extraer ficheros comprimidos en rar usando el gestor de ficheros comprimidos file-roller, necesita instalar este paquete de software. Por ser software no libre, no ha sido incluido en el disco de Guadalinex.

  • Instalar descompresor

Plugin Flash para Firefox:

Las animaciones y vídeos en formato Flash son muy habituales en internet. Para poder visualizarlas, necesita instalar este plugin para el navegador. Por no ser software libre, no ha sido incluído en el disco de Guadalinex.

  • Instalar plugin Flash
