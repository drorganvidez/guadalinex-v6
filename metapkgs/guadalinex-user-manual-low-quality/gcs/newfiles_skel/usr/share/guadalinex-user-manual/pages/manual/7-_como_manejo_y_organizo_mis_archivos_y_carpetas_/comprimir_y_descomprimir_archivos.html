Comprimir y descomprimir archivos

Antes de hablarte de cómo se lleva a cabo la coprensión y descompresión de archivos, nos gustaría aclararte la diferencia que hay entre Archivos comprimidos y Archivos empaquetados, ya que no es lo mismo y además hay programas que hacen una cosa, u otra, o las dos.

Cuando una serie de archivos o directorios han sido almacenados bajo un mismo archivo sin modificar su estructura , este archivo resultante, el que contiene todos los demás, es un Archivo empaquetado.

Y cuando una serie de archivos o directorios se almacenan bajo un mismo archivo pero reduciendo su estructura para que ocupe menos espacio de almacenamiento, el archivo resultante de menor tamaño es un Archivo comprimido.

En Guadalinex el programa que debes utilizar para comprimir y descomprimir archivos es el File Roller. No lo busques en el menú Aplicaciones del panel superior del escritorio, ya que este programa sólo se abrirá cuando hagas doble clic encima del icono de un archivo comprimido.

2- Comprimir archivos.

3- Descomprimir archivos.
