6- Barra de herramientas de marcadores

Ubicada debajo de la barra de navegación. Se utiliza para sacar del menú de marcadores aquellas direcciones de páginas web que más utilizas y tenerlas en forma de pestañas siempre a la vista. Para poder añadir pestañas a esta barra cuando agregues una web a marcadores asegúrate de que la guardas en la carpeta Barra de herramientas de marcadores.
