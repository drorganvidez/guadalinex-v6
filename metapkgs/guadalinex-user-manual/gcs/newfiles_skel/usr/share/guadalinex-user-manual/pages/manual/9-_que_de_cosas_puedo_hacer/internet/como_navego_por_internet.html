¿Cómo navego por Internet?

Si ya has contratado tu conexión de Internet con un proveedor, y has configurado correctamente el ordenador para activar dicha conexión, lo único que te queda es saber cómo entrar en Internet para empezar a navegar.

El acceso a Internet se realiza a través de un programa que se dedica a procesar las operaciones necesarias para establecer la conexión. Dicho programa se conoce normalmente como “navegador web”, y en Guadalinex el que utilizamos es el Mozilla Firefox.

Para abrir el navegador Firefox tienes que acceder al panel superior del escritorio y hacer clic en Aplicaciones→Internet→Navegador Web (Firefox). También puedes encontrar en el escritorio un icono del navegador y acceder a él haciendo doble clic.

Una vez que el navegador se ha abierto en tu pantalla, vamos a explicarte las diferentes herramientas del entorno de Firefox que te harán más fácil y más práctica tu navegación en Internet, aunque muchas de estas herramientas, a estas alturas ya las conoces o te suenan.

1- Barra de título.

2- Barra de menús.

3- Barra de herramientas.

4- Barra de navegación.

5- Barra de búsqueda web.

6- Barra de herramientas de marcadores.

7- Barra de estado.

8- Barra de búsqueda.

9- Barra de desplazamiento.

10- Privacidad y seguridad.

11- Ventanas emergentes.
