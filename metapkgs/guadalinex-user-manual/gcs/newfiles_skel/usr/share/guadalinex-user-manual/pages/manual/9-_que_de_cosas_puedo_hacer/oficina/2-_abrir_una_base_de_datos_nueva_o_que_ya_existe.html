Guadalinex V6 Guía de usuario

Al igual que otros programas, cuando entras en el OpenOffice Bases de datos, también te abre una ventana de asistente de forma automática, donde puedes hacer clic en la opción Crear nueva base de datos si ese es el caso, o la opción Abrir un archivo de base de datos existente si ya creaste una base de datos en otro momento y quieres recuperarla para consultas, modificaciones, añadir nuevos datos, etc… Para ello te aparecerá la ventana que has visto ya en más de una ocasión, que te permite buscar por los diferentes sitios de almacenamiento de tu ordenador para abrir archivos ya creados y guardados.

Sea cual sea la opción que elijas, cuando acabes haces clic en Finalizar, y te abrirá tu nueva base de datos después de que el programa te pida que la guardes antes, o te abrirá la que ya existía y quieres recuperar.
